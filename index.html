Cell Segmentation and Tracking in 4D images

Maize Detection and Classification in Multi-Spectral Images

This project centers on the deep learning-based detection and classification of healthy and unhealthy maize corn in multi-spectral images on a conveyor belt, employing selective search and a VGG-16-based CNN. The subsequent application of non-maximum suppression refines predictions, highlighting the feasibility of advanced deep learning for precise agricultural object detection.

U-Net for Retinal Blood Vessel Segmentation

This project employs the U-Net architecture for binary segmentation of retinal blood vessels using a labeled dataset of 20 images. Trained for 40 epochs, the U-Net demonstrates promising results in accurately segmenting retinal blood vessels.

Urban Spread Detection: Dubai Palm Islands (2000-2008)

This project focuses on detecting urban sprawl in Dubai's Palm Islands between 2000 and 2008, utilizing data from Google Earth Engine. The change map is computed through Canonical Correlation Analysis.
